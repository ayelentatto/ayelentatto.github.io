The CRUD concept is closely linked to digital data management. CRUD refers to an acronym in which the first letters of the four operations fundamentals of persistent applications in database systems:

Create (Create registers) Read bzw. Retrieve (read registers) Update (Update registrers) Delete bzw. Destroy (Delete registrers)

In a few words, CRUD summarizes the functions required by a user to create and manage data.
