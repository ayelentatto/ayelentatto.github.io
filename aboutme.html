IBM Full-Stack Software Developer Professional Certificate
From 9/01 to 31/03 of 2023
About this Professional Certificate This professional certificate will equip you with key skills and technical knowledge to jumpstart your career as a full-stack cloud-native application developer. Guided by IBM, you'll learn how to build cloud-based applications and work with the technologies that support them. This program consists of 10 courses with extensive instructional content, as well as hands-on exercises and projects designed to hone skills and help build your portfolio. No prior programming experience or cloud background is required to start this program. You'll gain skills with the tools and technologies that successful software developers will use to create, deploy, test, run, and manage Full Stack Cloud Native applications, giving you the practical skills to start a new career in an in-demand area. Courses in this program help you develop skill sets in many technologies, including: Cloud Fundamentals, HTML, CSS, JavaScript, GitHub, Node.js, React, Cloud Native Practices, DevOps, CI/CD, containers, Docker, Kubernetes, OpenShift, Istio, Python, databases, SQL, NoSQL, Django ORM, Bootstrap, application security, microservices, serverless computing, and more. After completing all the courses in the program, including the capstone project, they will have developed various applications using front-end and back-end technologies and deployed them on a cloud platform using cloud-native methodologies. Plus, this program is enhanced by ACE®—when you complete it, you can earn college credit.
