Courses

IBM Full-Stack Software Developer Professional Certificate
From 9/01 to 31/03 of 2023

The IBM FullStack Software Developer course is a comprehensive technology training program consisting of 12 essential technologies in the world of software development.

The program begins with an introduction to cloud computing, teaching the fundamental concepts and best practices needed to understand how it works and how to use it in software development projects. (view certificate)

Next, basic web development technologies with HTML, CSS and JavaScript are introduced. These skills are essential for any modern web developer, and are explored in depth so that students can build basic web applications from scratch. (view certificate)

Next, the program focuses on version control and collaborative teamwork using Git and GitHub. Students learn how to work efficiently in teams and collaborate on software projects using these tools. (view certificate)

The next section focuses on creating frontend applications with React, one of today's most popular JavaScript libraries. Students learn how to build interactive and reusable user interfaces for modern web applications. (view certificate)

Next, they delve into backend application development using Node.js and Express. These technologies are essential for building scalable and secure server-side applications. (view certificate)

The course also includes a section on Python for data science, AI and application development. Students learn the fundamentals of Python and how to apply them to data science, artificial intelligence and application development. (view certificate)

In addition, students work on a Python for AI and application development project, allowing them to apply what they have learned in a hands-on project.

The database section focuses on application development with SQL, databases and Django. Students learn how to build applications with secure and scalable databases.

Next, container technology is introduced with Docker, Kubernetes and OpenShift. Students learn how to create, deploy and scale containerized applications using these tools.

The next section focuses on microservices and serverless application development. Students learn how to create scalable and resilient applications using these technologies.

The course also includes a final capstone project on full stack cloud development, where students apply everything they have learned in a complete hands-on project.

Finally, students complete a full stack software developer skills assessment, which allows them to demonstrate what they have learned and receive an IBM certification in FullStack Software Developer.

In summary, the IBM FullStack Software Developer course is a comprehensive technology training program that provides students with the skills necessary to become full stack software developers. With a focus on cutting-edge technologies used today, students get a comprehensive, hands-on education that enables them to work on real-life software development projects.
